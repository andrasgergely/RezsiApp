# RezsiApp – Havi költség összesítő

Egyszerű grafikus program, amely a havi költségeimet összegzi megadott kategóriák alapján.

## Mit tud?
- Áram, gáz, víz, internet+TV megadása (internet+tv egybe vettem, mert általában egy szolgáltatótól van)
- Üzemanyag havi költsége
- Egyéb havi költség
- Havi összes költség kiszámítása
- A számítás időpontjának kiírása

## Használt modulok
- customtkinter (GUI)
- datetime
- saját modul: AA_module.py

## Futtatás
```
pip install customtkinter
python main.py
```
