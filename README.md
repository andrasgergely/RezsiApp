# RezsiApp

## Hallgató
Ábrahám András

## Feladat leírása
Grafikus program havi költségek összegzésére és alap statisztikai értékek kiszámítására.

## Modulok és a modulokban használt függvények
- AA_module: AA_Calculator, AA_total()
- statistics: mean(), median(), stdev()
- customtkinter, datetime

## Osztályok
- AA_Calculator

## Futtatás
Ha még nincs telepítve:

pip install customtkinter

Indítás:

python main.py
